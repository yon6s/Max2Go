# 开发记录

## 当前版本能力

- 单页 PHP 网站
- 登录保护
- DeepSeek API 调用
- 多项目管理
- 知识库管理
- 美兰湖竞品资料内置与可导入
- 8 个业务流程模块
- Excel 房源台账提取
- Excel 模板驱动价格测算
- 本次测算 Excel 下载

## 主要文件

```text
app/bootstrap.php              启动、配置、登录、JSON响应
app/prompts.php                AI提示词组装
app/database.php               MySQL连接与建表
app/project_store.php          项目管理
app/knowledge_store.php        知识库管理
app/pricing.php                备用价格测算公式
app/excel_template.php         Excel模板写入、重算、读取
public/index.php               页面入口
public/assets/app.js           前端交互
public/assets/max2go-data.js   从Excel提取的房源与J回正数据
public/api/generate.php        AI生成接口
public/api/knowledge.php       项目/知识库接口
public/api/price.php           价格测算接口
```

# 价格测算逻辑

## 业务目的

价格测算模块用于判断客户报价是否突破租决口径，并通过破底率对应 J 回正年数，辅助业务员控制报价空间。

核心问题：

- 客户合同面积和合同价格怎么报？
- 免租期、涨幅、物业收入和特殊事项对破底率影响多大？
- 当前条件下 J 回正年数是多少？
- 为了控制目标 J 回正年数，报价应当大致处在哪个区间？

## Excel 模板驱动

当前设计不把公式完全写死在网页中，而是保留 Excel 模板：

```text
app/templates/pricing_template.xlsx
```

网页负责：

- 选择 Excel 房源台账中的房源
- 写入租决面积、租决价格、客户合同面积、合同价格等输入
- 复制模板生成本次测算 Excel
- 尝试调用 LibreOffice 重算公式
- 读取破底率和 J 回正年数等结果

这样后续业务公式变化时，可以直接替换 Excel 模板。

## 关键公式口径

Excel 原始核心口径：

```text
综合破底率 = (租金差额 + 物业差额 + 特殊事项差额) / 租决租金总额
```

J 回正年数来自 Excel 右侧的破底率与 J 回正年数对照表。

## LibreOffice

服务器如安装 LibreOffice，可自动重算 Excel 公式：

```bash
apt install libreoffice
```

未安装时，系统会生成本次测算 Excel，并用后端备用公式输出关键指标。

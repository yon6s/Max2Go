# 比赛汇报思路

## 主题

```text
Max2Go｜MAX租赁AI工作台
把租赁顾问的个人经验，沉淀成可复制、可训练、可复盘的 AI 工作流。
```

## 展示主线

用一个模拟客户贯穿全流程：

```text
某科技研发企业，团队扩张到45人，需求350-500㎡，计划6月入驻，关注成本、形象、交付和竞品比较。
```

现场依次展示：

1. 线索与约访：勾选客户画像和成交信号，生成线索评分和约访话术
2. 房源匹配：根据面积、方位、空间配置生成分割建议
3. 价格测算：调用 Excel 房源和模板，计算破底率与 J 回正
4. 到访复盘：粘贴带看纪要，生成关注点、异议和下一步
5. 客户方案：生成三套面积/商务方案
6. 谈判助手：针对价格、免租期、竞品生成话术
7. 合同助手：生成合同字段清单和风险检查
8. 管理看板：沉淀线索、报价、丢单、竞品和复盘

## 评分点对应

### 落地性和转化价值

- 一线业务员日常就能使用
- 减少重复整理资料
- 提升报价测算和方案响应速度
- 帮助新人快速掌握项目口径

### 探索性和启发性

- 不只是写文案，而是把完整租赁流程搬进 AI 框架
- 知识库由业务自己维护
- Excel 模板公式保留，AI 和传统业务工具结合

### 深入度和专业性

- 接入项目竞品说辞
- 接入租决价格测算
- 输出破底率和 J 回正年数
- 合同模块明确不替代法务，只辅助业务草案和风险检查

## 核心表达

```text
AI不是替代租赁顾问，而是把租赁顾问的经验、项目资料、商务测算和客户复盘组织起来，让一线动作更快、更准、更可复制。
```

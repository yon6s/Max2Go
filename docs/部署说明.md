# 部署说明

## MAMP 本地部署

建议目录：

```text
/Applications/MAMP/htdocs/max2go
```

项目结构：

```text
/Applications/MAMP/htdocs/max2go/app
/Applications/MAMP/htdocs/max2go/public
```

访问地址：

```text
http://localhost:8888/max2go/public/
```

MAMP MySQL 默认配置通常为：

```php
'db_host' => '127.0.0.1',
'db_port' => '8889',
'db_name' => 'max_rental_ai',
'db_user' => 'root',
'db_password' => 'root',
```

## 宝塔 / LNMP 部署

建议网站目录：

```text
/www/wwwroot/panel_ssl_site
```

如果使用 `/max2go` 子路径，结构可为：

```text
/www/wwwroot/panel_ssl_site/app
/www/wwwroot/panel_ssl_site/max2go/index.php
/www/wwwroot/panel_ssl_site/max2go/api
/www/wwwroot/panel_ssl_site/max2go/assets
```

注意：

- `app/config.php` 不要上传到公开仓库
- 若出现 `open_basedir` 错误，需要允许 PHP 读取上级 `app` 目录
- 数据库信息写入 `app/config.php`
- Excel 自动重算建议安装 LibreOffice

## 配置文件

复制：

```bash
cp app/config.example.php app/config.php
```

配置：

```php
'app_password' => '访问密码',
'deepseek_api_key' => 'DeepSeek API Key',
'deepseek_base_url' => 'https://api.deepseek.com',
'deepseek_model' => 'deepseek-chat',

'db_port' => '3306',
'db_name' => '数据库名',
'db_user' => '数据库用户',
'db_password' => '数据库密码',
```

## 常见问题

### AI接口返回 Service is too busy

这是模型服务繁忙，不是网站故障。建议后续增加备用模型。

### 知识库报 Unexpected token '<'

表示接口返回了 HTML 错误页，不是 JSON。通常是 PHP 报错、路径没覆盖完整或未登录。

### 价格测算未完全按 Excel 公式变化

检查服务器是否安装 LibreOffice。未安装时会使用备用公式计算关键指标。

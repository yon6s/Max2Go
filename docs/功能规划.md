# 功能规划

## 当前 8 个业务模块

1. 线索与约访  
   汇总客户画像、成交信号、需求采集、推进紧急度，生成线索评分、约访话术和到访准备。

2. 房源匹配与分割建议  
   根据面积、行业、方位、空间配置、楼层条件，生成推荐房源、分割建议和带看顺序。

3. 价格测算与报价空间  
   调用 Excel 房源台账和测算模板，输入合同面积、合同价格、免租期、涨幅、特殊事项，输出破底率和 J 回正年数。

4. 到访录音复盘  
   粘贴录音转文字或带看纪要，提取客户关注点、异议、意向等级和下一步动作。

5. 客户方案生成  
   生成客户版空间方案、商务条件、综合成本对比和方案 PPT 大纲。

6. 谈判话术助手  
   根据客户异议、竞品、审批底线，生成回应话术、条件交换建议和上报口径。

7. 合同草案助手  
   根据合同关键条款生成字段清单、风险检查和业务草案提醒。

8. 管理看板与复盘沉淀  
   从部门管理角度总结线索、到访、报价、签约、丢单、竞品和房源热度。
